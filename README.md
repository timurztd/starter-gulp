# push-starter-gulp

## Installation

```bash
npm install
```

## Run dev server

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Usage
### Adding css vendors
To add css vendors update `cssVendors` in `gulp\paths.js` and uncomment `vendors.css` in `src\templates\partials\_head.html`
