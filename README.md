# push-starter-gulp

## Installation

```bash
npm install
```

## Run dev server

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Usage
### Adding css vendors
To add css vendors update `cssvendors.js` in `/gulp` folder and restart dev server
